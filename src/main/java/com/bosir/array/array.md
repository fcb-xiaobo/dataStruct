数组介绍


1）什么是数组

数组（Array）是一种线性表数据结构。
它用一组连续的内存空间，来存储一组具有相同类型



线性结构

线性表就是数据排成像一条线一样的结构。每个线性表上的数据最多只有前和后两个方向。
其实除了数组，链表.队列、栈等也是线性表结构。而与它相对立的概念是非线性表，
比如二叉树、堆、图等。之所以叫非线性，是因为，在非线性表中，数据之间并不是简单的前后关系。
